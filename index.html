Frequently Asked Questions

常見問題

預約流程係點？+
選擇 Package → 選擇拍攝日期和時間 → 落訂確認
幾多週拍攝最理想？+
建議 28-34 週拍攝，因為 36 週後肚子急速增大，部分媽媽亦會出現水腫、腰背疼痛等情況。
有冇化妝服務？+
Package 已經包咗化妝服務（化妝 + Set 頭）。
衣服要準備嗎？+
我哋有多款衣服提供畀孕婦媽媽配搭選擇，爸爸衣服需自備。
需要準備道具嗎？+
可以準備 BB 超聲波相、衫仔鞋仔、自製相簿或對你哋有紀念價值的小道具，令相片更加獨一無二。
可以帶毛孩一起拍攝嗎？+
毛孩都係家人，非常歡迎帶埋佢哋一齊嚟影相。
如果拍攝當日天氣惡劣情況？+
遇上黑雨 / 8 號風球或以上，可安排免費改期一次。如需改期，請於化妝時間前最少兩小時通知我們。
初生 BB 幾時拍攝最好？+
建議最佳拍攝期為出生後 7-14 日，因為呢個階段 BB 關節及肌肉仍非常柔軟，另外大部分時間處於深度睡眠，更容易配合拍攝。初生 BB 樣子變化很快，所以千祈唔好錯過呢個珍貴時刻。
預約流程係點？+
選擇 Package → 落訂確認 → 拍攝日期和時間可以預早約或等 BB 出世再約。
有沒有上門服務？+
我哋有提供上門服務或可以嚟我哋 Studio（荃灣 TML TOWER）。
上門和 Studio 拍攝有什麼分別？+
有部份造型道具比較大型，較難帶上門，所以只限 Studio 拍攝。
Studio 可以最多容納幾多人拍攝？+
大約 20 人左右都可以。
可以帶毛孩一起拍攝嗎？+
毛孩都係家人，非常歡迎帶埋佢哋一齊嚟影相。
寵物攝影常見問題+
內容更新中，歡迎透過 Instagram DM 查詢寵物拍攝詳情。
