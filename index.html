Frequently Asked Questions

常見問題

預約流程係點？+
選擇 Package → 選擇拍攝日期和時間 → 落訂確認
幾多週拍攝最理想？+
建議 28-32 週拍攝，因為 36 週後肚子急速增大，部分媽媽亦會出現水腫、腰背疼痛等情況。
有冇化妝服務？+
Package 已經包咗化妝及髮型造型服務。
衣服要準備嗎？+
我哋有多款衣服提供畀孕婦媽媽配搭選擇，爸爸衣服需自備。
需要準備道具嗎？+
可以準備 BB 超聲波相、衫仔鞋仔、自製相簿或對你哋有紀念價值嘅小道具，令輯相更加獨一無二。
可以帶毛孩一起拍攝嗎？+
毛孩都係屋企人，非常歡迎帶埋佢哋一齊嚟影相。
如果拍攝當日天氣惡劣？+
黑雨 / 8 號風球或以上，可安排免費改期一次。如需改期，請於化妝時間前最少兩小時通知我哋。
你哋影樓喺邊度？係咪荃灣？+
我哋 Studio 位於荃灣 TML Tower，交通方便，大廈內設有停車場。
有冇韓式大肚相？+
有，孕婦攝影（大肚相 / 孕婦寫真）提供自然風、黑白風及韓式風格，可按你喜好配搭。
孕婦相、BB相、全家福大概幾錢？+
孕婦攝影由 HKD 2,380 起、初生及百日照由 HKD 3,800 起、家庭攝影（全家福）由 HKD 1,800 起。實際收費視乎人數、場景及需求，詳情請參考價目表或 IG DM 查詢。
初生 BB 幾時拍攝最好？+
建議最佳拍攝期為出生後 7-14 日，因為呢個階段 BB嘅關節同肌肉仲係好柔軟，另外大部分時間處於深度睡眠，更容易配合拍攝。初生 BB 個樣變得好快，所以千祈唔好錯過呢個珍貴時刻。
預約流程係點？+
選擇 Package → 落訂確認 → 拍攝日期同時間可以預早約或等 BB 出世再約。
有沒有上門服務？+
我哋有提供上門服務或可以嚟我哋 Studio（荃灣 TML TOWER）。
上門和 Studio 拍攝有什麼分別？+
有部分造型道具比較大型，較難帶上門，所以只限 Studio 拍攝。
初生 BB 相上門影定去影樓好？+
兩種都可以。上門影 BB 方便初生寶寶，唔使舟車勞頓；影樓（荃灣 Studio）則有更多造型場景同道具選擇。
有冇百日宴相、周歲抓周或 Cake Smash？+
有，BABYG 提供百日照（百日宴相）、一歲生日照、周歲抓周及 Cake Smash 拍攝，紀錄 BB 每個成長里程碑。
Studio 可以最多容納幾多人拍攝？+
大約 20 人左右都可以。
可以帶毛孩一起拍攝嗎？+
毛孩都係屋企人，非常歡迎帶埋佢哋一齊嚟影相。
荃灣有冇影全家福 / 親子相？+
有，我哋荃灣 Studio 提供家庭攝影（全家福 / 親子相 / 三代同堂）。
